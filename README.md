# Sonieo Production

Premium personal video editing portfolio built with Next.js App Router and Tailwind CSS.

## Stack

- Next.js 16
- React 19
- Tailwind CSS
- TypeScript

## Features

- Single-page conversion-focused portfolio layout
- Sticky responsive navigation
- Hero section with local intro video
- Portfolio showcase with video modal
- Social proof, case studies, testimonials, FAQ, and final CTA
- SEO metadata, Open Graph, and structured data
- Responsive dark premium UI

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Production Preview

Build the app:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Deploy

Recommended: Vercel

1. Import this GitHub repo into Vercel.
2. Framework preset: `Next.js`
3. Build command: `npm run build`
4. Output setting: default Next.js output
5. Add environment variable:

```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.vercel.app
```

6. Deploy
